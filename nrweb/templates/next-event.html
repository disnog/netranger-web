Call for Presentations

For each quarterly meeting, we wish to select up to 2 lightning talks (~10 minutes) and up to 1 Technical Presentation (~25 minutes) from server participants. These should be presentations on things you've been working on, examples of an interesting configuration, demonstration of a network automation approach or product, or even a talk about career growth strategy, education, etc. There are few limitations as long as it's relevant to network engineering. Open source and server member developed software and ideas will be given priority to vendor presentations. If you are interested in presenting, please contact Jason.
